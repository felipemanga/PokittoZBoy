◆ audio_IRQ()

void Pokitto::audio_IRQ ( )

Sound class. The Sound class is an API-compatible version of the Gamebuino Sound API by Aurelien Rodot. Because it is a class consisting of only static members, there is only 1 instance running, no matter how many Sound classes are declared (see example below). This means sound can be used through a simple Sound class object or as a member of the Core class.
